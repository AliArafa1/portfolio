## ✨ Key Features
- **Bilingual Support (English & Arabic):** Full localization with layout flipping (RTL) and typography adjustments for Arabic using the Cairo font.
- **Project Showcase:** Interactive project detail modals with glassmorphism effects, key features, and technology tags.
- **Certificate Gallery:** A comprehensive carousel slider displaying 47 professional certificates including Cisco (CCST, CCNA), Cybersecurity, IoT, IBM AI, and more.
- **Dynamic Contact Form:** Working contact form connected via Web3Forms API.
- **Modern UI/UX:** Responsive design with dynamic matrix/network background animations, smooth scrolling, and hover effects.

## 🛠️ Technologies Used
- HTML5 / CSS3 (Custom variables, flexbox, glassmorphism)
- Vanilla JavaScript
- Web3Forms API for backend form handling
